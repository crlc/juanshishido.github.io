General Linear Models

Simple Linear Regression

A line can be described by the equation \(y = mx + b\), where \(b\) is the \(y\)-intercept and \(m\) is the slope. Simple linear regression uses this idea to describe the relationship between two variables. The process uses the data to find the line that "best" fits it.

Because each observation has its own \(x\) and \(y\) value, we use subscripts to denote them. In addition, unless the data form a single straight line, we will need to introduce an additional concept—errors.

When we fit a line to our data, there will be points above and below it. The differences between these points and the line is referred to as the "error." To capture this, we add a new term to our equation. $$y_i = \beta_0 + \beta_1 x_i + \epsilon_i$$ In this form, in which the terms have been slightly reordered from the equation above, \(\beta_0\) represents the \(y\)-intercept, \(\beta_1\) represents the slope, and \(\epsilon_i\) represents the error. Notice that the \(y\), \(x\), and \(\epsilon\) terms all have subscripts, but the \(\beta\)s do not. This implies that there is a single \(y\)-intercept and slope.

While we have specified the relationship we believe the data have—linear in this case—the model is not fully specified. For this, we must be explicit about our assumptions of \(\epsilon\). We can, for example, assume that \(\epsilon\) are independent and identically distributed with a zero mean and \(\sigma^2\) variance.

Matrix Form

It is useful to specify our model in matrix form. We can think of \(y_i\), \(x_i\), and \(\epsilon_i\) as being vectors with \(n\) values, where \(i \in 1,\ldots,n\). We can write \(y_i\) as \(\vec{y}\). Using this notation, the \(\beta\) values are scalars. However, with what we've described thus far, \(\beta_0\) doesn't have a vector of values to be multiplied with. So, we add a vector of ones of length \(n\).

Our system of \(n\) equations can be represented as follows. \begin{split}\left[\begin{matrix}\epsilon_{1}\\ \vdots\\ \epsilon_{n}\end{matrix}\right]\end{split}
